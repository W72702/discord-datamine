<h1 id="h_01H0TCJ3ZR5K7A387D2G661Z9S"><strong>Creating a Server Template</strong></h1>
<h3><strong>Note: This feature is currently only available on the desktop and browser app. If you’re on the mobile app, you sadly won’t be able to create a new Server Template at this point in time.</strong></h3>
<p><strong>1.</strong> To create a Server Template, you will need to be the owner of the server or have a role with the <strong>Manage Server</strong> permission enabled: </p>
<p class="wysiwyg-text-align-center"><img style="height: 91px; width: 412px;" src="https://support.discord.com/hc/article_attachments/14711163397399"></p>
<p><strong>2.</strong> If you meet either requirements, navigate to <strong>Server Settings</strong> by selecting the dropdown menu next to the server name or by right-clicking the server name itself:</p>
<p class="wysiwyg-text-align-center"><img style="height: 329px; width: 256.0281876238714;" src="https://support.discord.com/hc/article_attachments/14711147606807"></p>
<p>3. Within Server Settings, select the <strong>Server Templates</strong> tab:</p>
<p class="wysiwyg-text-align-center"><img style="height: 386px; width: 257.0134986081202;" src="https://support.discord.com/hc/article_attachments/14711147646743"></p>
<p><strong>4.</strong> After being redirected to <strong>Server Templates</strong>, you can give your template a name and short description. After you fill out the template title and description, press <strong>Generate Template</strong> to create a template link. </p>
<p class="wysiwyg-text-align-center"><img style="height: 228.3060806911387; width: 580px;" src="https://support.discord.com/hc/article_attachments/14711132300823"></p>
<p>6. Once you’ve generated a link, simply select <strong>Copy</strong> and you are ready to share your Server Template!</p>
<p class="wysiwyg-text-align-center"><img style="height: 107.98886827458257; width: 534px;" src="https://support.discord.com/hc/article_attachments/14711147670551"></p>
<h2 id="h_01H0TCJDWN13DFN9FTP502E13Q"><strong>Sync Server Templates</strong></h2>
<p>Server Templates do not automatically update when you make changes to the original server's set-up. That means if the Server Template isn't synced, you won't see the new changes when using the template!</p>
<p>However, a notification will pop up by your server’s name when you make changes to your server:</p>
<p class="wysiwyg-text-align-center"><img style="height: 106px; width: 303.2177419354839;" src="https://support.discord.com/hc/article_attachments/14711147655703"></p>
<p>In order to sync your Server Template, go to Server Settings &gt; Server Templates &gt; and press the <strong>Sync Template </strong>button to update the template!</p>
<p class="wysiwyg-text-align-center"><strong><img style="height: 319px; width: 540px;" src="https://support.discord.com/hc/article_attachments/14711178233495"></strong></p>
<h2 id="h_01H0TCJKQ3W0VF8TPMS5PN548P"><strong>Deleting Server Templates</strong></h2>
<p>Server Template links can also be deleted by going to server settings &gt; Server Templates &gt; <strong>Delete Template </strong></p>
<p class="wysiwyg-text-align-center"><strong><img style="height: 280px; width: 540px;" src="https://support.discord.com/hc/article_attachments/14711116430743"></strong></p>
<h2 id="h_01H0TCJXK9B5N6YQZBESF7KBEB">Server Template Creation Errors</h2>
<p>If you run a large server or a server that uses a large number of Role Permissions overwrites, you may run into the “<strong>Maximum number of channel permission overwrites reached (1000)</strong>” error. </p>
<p> </p>
<p>This indicates that you have added over 1,000 Role Permissions overwrites - these are permissions that are added per-channel, rather than within your Server Settings.</p>
<p> </p>
<p> </p>
<p class="wysiwyg-text-align-center"><img style="height: 200px; width: 329px;" src="https://support.discord.com/hc/article_attachments/14711178225431"></p>
<p> </p>
<p>If you run into this error, you’ll need to reduce the number of Role Permissions overwrites in your Server before you are able to create a Server Template.</p>
<p> </p>
<h3>
    <strong><span style="color: #434343;" data-darkreader-inline-color="">Note:</span></strong><span style="color: #434343;" data-darkreader-inline-color=""> This limit does apply retroactively, so we recommend keeping tabs on the number of permission overwrites you are adding to your server to ensure that your friends are able to use your Server Template as well.</span>
</h3>
<p> </p>
<p class="wysiwyg-text-align-center"><span style="color: #434343;" data-darkreader-inline-color=""><img style="height: 648px; width: 611.8885448916409;" src="https://support.discord.com/hc/article_attachments/14711147755159"></span></p>
<h1 id="h_01H0TCK5PHAH4ZHZA1WQX2T3FG"><strong>Using Server Templates</strong></h1>
<p>Server Templates must be generated by a user who has the <strong>Manage Server</strong> permission. Once a Server Template has been generated, you can share it in chat and it will look like the image below: </p>
<p class="wysiwyg-text-align-center"><img style="height: 219px; width: 598px;" src="https://support.discord.com/hc/article_attachments/14711194349079"></p>
<p>Before you create a Server using a Server Template, you can preview the template!</p>
<p class="wysiwyg-text-align-center"><img style="height: 377px; width: 624px;" src="https://support.discord.com/hc/article_attachments/14711116453143"></p>
<p>If you’re satisfied with the preview of the server template, you can rename the server and upload a new server image from the preview window.</p>
<p>Once you select <strong>Create</strong>, brand new servers can be created with all the preset channels, roles, and permissions using the server template. </p>
<p class="wysiwyg-text-align-center"><img style="height: 327px; width: 537px;" src="https://support.discord.com/hc/article_attachments/14711147697943"></p>
<h3><strong>Note: If the Template creator deletes the Server Template, the link will no longer work.</strong></h3>
