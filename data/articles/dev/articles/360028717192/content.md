<p>For developers, all Applications, Teams, &amp; Servers have a multi-digit ID that is helpful to our developer support team if you're in need of assistance. Here we'll go over how to find those! </p>
<h2>Enable Developer Mode</h2>
<p>Head into <strong>User Settings &gt; Appearance &gt; Advanced</strong> &amp; enable <strong>Developer Mode.</strong></p>
<p><img src="https://support-dev.discord.com/hc/article_attachments/360029908211/1.png" alt="1.png"></p>
<h2>Application ID</h2>
<p>The Application ID, also known as the <strong>Client ID</strong>, can be found in the Developer Portal in <strong>General Information</strong>. </p>
<p><img src="https://support-dev.discord.com/hc/article_attachments/360029911991/2_.png" alt="2_.png"></p>
<p>You can also find the Application ID by <strong>right-clicking</strong> on the game in your <strong>Library</strong>.</p>
<p><img src="https://support-dev.discord.com/hc/article_attachments/360039180751/3.png" alt="3.png"></p>
<h2>Team ID</h2>
<p>To get the <strong>Team ID</strong> you'll want to head into <strong>Developer Portal &gt; Teams</strong> tab &amp; go to the <strong>URL</strong>. The Team ID will be located here. </p>
<p><img src="https://support-dev.discord.com/hc/article_attachments/360029918431/4_.png" alt="4_.png"></p>
<h2>Server ID</h2>
<p>Right-click the server icon &amp; select <strong>Copy ID</strong>. Double check if you have Developer Mode enabled if you're not seeing a Copy ID option. </p>
